三、Sublime Text 4汉化

  1. 点击 Tools—Install Package Control（安装包控件比较慢，并且没有反应，等待数分钟后会有弹窗）
  2. 点击确定按钮
  3. 菜单点击Preferences – Package Control

  1. 输入 Install Package（安装包控件比较慢，并且没有反应，等待数分钟后会有弹窗）

  1. 输入 ChineseLocalzations 可见中文包，选中即可安装

四、中文乱码问题

  1. 点击菜单栏「Preferences」下的「Package Control」
  2. 进入Package Control，选择「Install Package」命令
  3. 进入安装界面，输入并选择「ConvertToUTF8」
  4. 成功安装「ConvertToUTF8」之后，乱码问题就解决了

五、更改显示字体

  1. 点击菜单栏「Preferences」下的「settings」
  2. 弹出如下配置文件窗口

  1. 参照系统默认配置，找到更改字体这一项，复制到用户这一栏的大括号内（ {} ）进行编辑

  1. 在 " " 中填入想要更改的字体(注，字体必须为电脑已安装的字体，字体名字建议前往控制面板查看，不要输错，否则无反应)
    这里我使用的是JetBrains家的一款字体 JetBrains Mono ,更改字号为12号

六、设置底部状态栏显示当前编码格式

  1. 点击菜单栏「Preferences」下的「settings」
  2. 弹出配置文件窗口
  3. 找到对应位置（第611行），在用户自定义栏进行编辑

  1. 可以看到，底部状态栏显示当前编码格式为UTF-8

七、取消自动更新提示

  1. 找到Preferences -> Settings
  2. 键入
    “update_check”:false
  3. 保存即可
